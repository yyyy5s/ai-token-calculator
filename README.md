# ai-token-calculator

这是一个基于浏览器的 AI 模型费用计算器，专注于 token 成本与缓存命中率的可视化估算。

主要功能：

- 支持多种模型定价类型：固定单价、按输入分档、按输出分档。
- 支持自定义模型与自定义币种管理。
- 提供单次 / 每日 / 每月成本估算，支持 USD 与 CNY 双币种显示。
- 自动获取实时汇率，并在界面语言切换后保持本地化展示。
- 支持模型对比，便于横向评估不同 AI 模型的费用差异。

适用场景：

- 需要估算不同 AI 模型调用成本的产品经理或开发者。
- 希望对比模型定价策略（例如缓存命中率、输入/输出 token）对总成本影响的用户。
- 希望在本地浏览器端快速测试 AI 服务费用的场景。

项目结构：

- `index.html`：主应用页面。
- `src/`：核心业务逻辑、渲染、存储和汇率缓存代码。
- `data/`：内置模型、币种和多语言文本配置。
- `sw.js`：服务工作线程，用于缓存静态资源。
